# 해한건설 출퇴근 앱 이용 안내

> 배포: 카카오톡 단체방 / 현장 공지
> 대상: 파일럿 참여 근로자 전원

---

## 출퇴근 앱 시작 방법

**접속 주소**: https://attendance.haehan-ai.kr

---

### 1단계 — 앱 접속 및 등록

1. 휴대폰 브라우저에서 **https://attendance.haehan-ai.kr** 접속
2. 전화번호 입력 후 **"로그인"** 버튼 누르기
3. "기기 승인 대기 중" 메시지가 나오면 **관리자 승인을 기다리세요**
4. 승인 완료 문자 또는 관리자 안내 후 다시 로그인하면 사용 가능

> 최초 1회만 승인이 필요합니다. 이후 로그인은 바로 됩니다.

---

### 2단계 — 출근

1. 현장 내에서 **출근 QR을 스캔**하거나 QR 아래 링크를 직접 눌러주세요
2. GPS가 자동으로 위치를 확인합니다 (현장 밖에서는 출근되지 않습니다)
3. "출근 완료" 메시지가 나오면 정상 처리된 것입니다

---

### 3단계 — 현장 이동

1. 다른 현장으로 이동한 경우 **이동한 현장의 QR을 스캔**하세요
2. 이동 기록이 자동으로 처리됩니다
3. 이동하지 않은 경우 이 단계는 생략해도 됩니다

---

### 4단계 — 퇴근

1. 퇴근 전 **마지막으로 있었던 현장의 QR을 스캔**하세요
2. "퇴근 완료" 메시지가 나오면 정상 처리된 것입니다
3. **퇴근 스캔을 잊으면 미퇴근 처리됩니다** — 반드시 스캔 후 귀가하세요

---

## 꼭 지켜주세요

| 항목 | 내용 |
|------|------|
| GPS 허용 | 앱 실행 전 휴대폰 위치 권한(GPS)을 허용해 주세요 |
| 현장 내 스캔 | QR은 반드시 현장 안에서 스캔해야 합니다 |
| 퇴근 필수 | 당일 퇴근 스캔을 빠뜨리면 다음날 자동 미퇴근 처리됩니다 |
| 기기 1대 | 1인 1기기 원칙입니다. 기기를 바꾼 경우 관리자에게 알려주세요 |

---

## 문제가 생기면

| 증상 | 조치 |
|------|------|
| 로그인이 안 됩니다 | 전화번호 확인 후 재시도, 안 되면 관리자에게 연락 |
| 기기 승인 대기 화면이 계속 나옵니다 | 현장 관리자에게 즉시 알려주세요 |
| QR 스캔 후 오류 메시지 | GPS 허용 여부 확인, 현장 안인지 확인 후 재시도 |
| 퇴근을 못 했습니다 | 당일에는 관리자에게 연락, 다음날도 가능하면 보정 요청 |
| 앱이 열리지 않습니다 | 브라우저(크롬, 사파리)로 직접 주소 입력 |

**현장 관리자 연락처**: _(현장별 연락처 기재)_

## 자주 묻는 질문

**Q. 앱을 설치해야 하나요?**
A. 아닙니다. 브라우저에서 접속하면 됩니다. 설치 없이 사용 가능합니다.

**Q. 와이파이가 없어도 되나요?**
A. 데이터 통신(LTE/5G)이 필요합니다. 신호가 약하면 실패할 수 있습니다.

**Q. QR을 모르는 경우 어떻게 하나요?**
A. 현장 내에 부착된 QR 인쇄물을 사용하거나 관리자에게 문의하세요.

**Q. 출근을 두 번 스캔하면 어떻게 되나요?**
A. 당일 동일 현장 출근은 1회만 처리됩니다. 중복 스캔은 무시됩니다.
