# 현장 출퇴근 앱 사용 안내

> 해한건설 QR 출퇴근 시스템 | 근로자용

---

## 앱 사용 전 준비

1. 스마트폰 **위치(GPS) 권한**을 허용하세요.
2. 관리자에게 **이름 + 휴대폰 번호 + 소속**을 전달하여 등록을 요청하세요.
3. 관리자가 기기를 승인해야만 출근이 가능합니다.

---

## 첫 로그인 방법

```
① 앱 접속
② 휴대폰 번호 입력
③ 문자(SMS)로 받은 인증번호 6자리 입력
④ 관리자 승인 대기
⑤ 승인 완료 후 QR 출근 가능
```

> 인증번호는 5분 안에 입력해야 합니다.
> 승인은 관리자가 직접 처리하므로, 승인 전에는 출근이 되지 않습니다.

---

## 출근 방법

```
① 현장 도착
② 앱 실행
③ 현장에 붙어있는 QR 스캔
④ "출근 완료" 확인
```

> 현장 반경 100m 안에서만 출근이 인정됩니다.
> GPS가 꺼져 있으면 출근이 되지 않습니다.

---

## 다른 현장으로 이동할 때

```
① 새 현장 도착
② 앱 실행
③ 새 현장 QR 스캔
④ "이동 완료" 확인
```

> 이동 이력은 자동으로 기록됩니다.
> 마지막으로 이동한 현장이 오늘의 인정 현장이 됩니다.

---

## 퇴근 방법

```
① 퇴근 전
② 앱 실행
③ 현재 현장 QR 스캔
④ "퇴근 완료" 확인
```

> 반드시 퇴근 전에 QR을 스캔해야 합니다.
> 퇴근 스캔을 하지 않으면 미퇴근으로 처리될 수 있습니다.

---

## 자주 묻는 질문

| 상황 | 해결 방법 |
|------|---------|
| QR이 안 찍혀요 | 카메라 권한 확인 후 재시도 |
| 출근이 안 돼요 | GPS 켜기 → 현장 반경 확인 → 관리자 문의 |
| 퇴근을 못 했어요 | 즉시 관리자에게 연락 (당일 처리 가능) |
| 앱이 안 열려요 | 앱 재시작 → 네트워크 확인 |
| 기기를 바꿨어요 | 관리자에게 기기 변경 요청 필요 |

---

## 문의

출퇴근 관련 문제는 **즉시 현장 관리자에게 연락**하세요.
당일 처리가 원칙이며, 다음날 이후 소급 처리는 제한될 수 있습니다.
